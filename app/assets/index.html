installing

  • requires latest node.js (v8.2.1)
  • brunch npm install -g brunch
  • git clone https://github.com/neuthral/this_repository.git - this will create a local copy of the source code for the app
  • cd node-js-getting-started - change directory into the local source code repository
  • npm install - install nessecary packages
  • npm start - starts local web server for development
